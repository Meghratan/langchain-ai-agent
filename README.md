# Generative AI Agent Chatbot using LangChain, Gemini API and Streamlit (Deployed)

This project is an AI-powered chatbot built using LangChain Agents with tool calling.

## Features

- LangChain Agent with tool support
- Gemini API integration
- SerpAPI for real-time search
- Chat history memory
- Streamlit Chat UI
- Deployed on Streamlit Cloud

## Live Demo

https://langchain-ai-agent-l.streamlit.app

## Tech Stack

- Python
- LangChain
- Gemini API
- SerpAPI
- Streamlit

## Run locally

pip install -r requirements.txt  
streamlit run agent_app.py
